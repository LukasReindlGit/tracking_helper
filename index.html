This page must be served over HTTP. Opening index.html directly (file://) prevents the app script (ES modules) from loading, so buttons will not work. From this folder run: python3 -m http.server 8080 then open http://localhost:8080/

Storage consent

This app can save your topics and time entries in your browser’s local storage so they persist after you close the tab. We do not use analytics or third-party tracking. If you decline, everything stays in memory only until you refresh the page.

German / EU: by accepting, you agree to this use of device storage for the tracking helper.

Tracking Helper

Per-row stopwatch for tickets — time shows as h:mm:ss while running; you can still type decimal hours (e.g. 5,3) or h:mm:ss; Copy still uses decimal hours for Salesforce or elsewhere.

Time tracking

Each row: drag the handle to reorder, optional full URL (used as-is in the scaled table link), ticket name, Scale (uncheck to keep actual hours in the scaled chart—e.g. a fixed meeting), duration, play / pause, copy, hide (or unhide when shown), and delete. Use “Show hidden rows” to see hidden tickets again.
